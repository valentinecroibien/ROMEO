Vous trouverez dans ce dossier 4 documents. 

Un fichier solveurLU.c permettant de résoudre un système linéaire tridiagonal. Pour exécuter ce document vous n’avez qu’à écrire les commandes : 
					
					gcc -ansi -pedantic solveurLU.c -lm
					./a.out

Un fichier solveurMDF.c permettant de résoudre un problème elliptique à partir de la méthode des différences finies. Pour exécuter ce document vous n’avez qu’à écrire les commandes : 
					
					gcc -ansi -pedantic solveurMDF.c -lm
					./a.out

Finalement, si vous souhaitez entrer plus en détail dans la résolution du problème elliptique par  la méthode des différences finies. Vous pouvez exécuter le fichier main.py. avec la commande run main.py
Après avoir lancé le programme, vous pourrez choisir les nombres d’intervalles à utiliser.
